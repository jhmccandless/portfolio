Death in the Afternoon

Notable Technologies Used:

  • Axios(API-handling)
  • Vanilla Javascript
  • HTML/CSS

This application is designed as a point-of-sale system with a set group of cocktails imported from an API. One chooses one of the six drinks and is them redirected to the drink information page. After seeing the drink details and confirming this is the desired drink, you can add the drink and go to the cart page where the order is totaled. As a group project, jobs were divided out from person to person with my main job focusing on the second, drink detail page. I also had a large responsibility in the way the application visually looks. Other, smaller jobs included helping the team with problem solving issues as well as leading meetings to be sure tasks were handled and completed. We used not only the fundamental Javascript, CSS, and HTML, but also axios to handle the API calls as well as local storage to pass the information form page to page as the user clicks through.
